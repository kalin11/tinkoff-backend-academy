Iterator - интерфейс, который реализует все коллекции из Collection Framework.
For each цикл - позволяет пробежаться по всем элементам коллекции.

В целом, эти инструменты об одном и том же - они позволяют проитерироваться, но есть ситуации, когда нужно использовать что-то одно.
Например:

1) итерируясь с помощью итератора можно удалять элементы коллекции, а если итерироваться с помощью for each, то вылетит Concurrent Modification Exception
2) с помощью итератора нельзя проитерироваться по всем элементам статического массива (например, int[] a = {1,2,3}, так как Iterator есть только у коллекций)
3) с помощью for each можно делать вложенные циклы и каждый раз во внешнем цикле итерация будет начинаться с нулевого элемента. В то время как с помощью итератора мы так сделать не сможем, так как итератор будет оставаться на прежнем месте. В какой-то момент мы просто словим NoSuchElementException.
